### Project Setting
$ npm install -g create-react-app
$ create-react-app peach_web

Router 
	$ yarn add react-router-dom
	$ yarn add cross-env --dev

Naver Maps Package
    $yarn add react-naver-maps

react-carousel
    $ npm i @brainhubeu/react-carousel

react-range
    $ yarn add react-range

Axios
    $yarn add axios

Cros
    $yarn install cros
